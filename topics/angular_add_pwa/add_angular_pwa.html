Add Angular PWA

This is where the magic happens. Once you have your existing or Angular CLI generated template app, we can now start adding the PWA features like service worker, manifest and icons.



Add @angular/pwa to the project

Once you're in the directory of the project you just made, add @angular/pwa.

ng add @angular/pwa

What the command does is modify your project by adding icons, manifest, service worker and theme color to your project.
